# STAR SURVIVER：アルファ焦点仕様

## 1. アルファ版の目的

アルファ版の判定基準は、次の2点に集約する。

1. 慣性移動しながら戦うこと自体が気持ちいいか。
2. 母艦から出撃し、任務を終えて、母艦へ帰還する体験が成立しているか。

この2点が成立していれば、他の拡張要素は未実装でもよい。

## 2. アルファで検証するコアループ

### 必須ループ

`母艦` -> `出撃` -> `戦闘` -> `任務達成` -> `帰還` -> `着艦` -> `報酬精算`

### ループの役割

- 母艦はメニューではなく、出撃の起点と帰還の終点である。
- 戦闘は「倒すこと」よりも「慣性を扱いながら生き残ること」を主眼に置く。
- 帰還は単なる離脱ではなく、速度制御と着艦精度を要求する緊張フェーズである。

## 3. アルファで残す要素

- 慣性移動、加速、減速、旋回
- ブースト
- ヒート管理
- 自動射撃または自動照準を含む基本戦闘
- 母艦の位置と着艦判定
- HUDの最低限の可視化
- Cielo の通信による状況説明

## 4. アルファで後ろへ送る要素

- SLG / 前線ゲージ / 広域戦況の動的変化
- 非同期 GvG
- 季節制やランキングのメタゲーム
- 複雑な経済圧迫や借金ループの拡張
- 大規模NPC群や特殊ミッションの網羅

## 5. 受け入れ条件

アルファ版は、少なくとも次の状態を満たしたときに「成立」と判断する。

1. プレイヤーが慣性を理解し、思った通りに速度を制御できる。
2. 戦闘中にブーストやヒートの判断が発生する。
3. 出撃から帰還までが1つの体験として完結する。
4. 着艦成功がプレイヤーに達成感を与える。

## 6. 構想の扱い

本作の将来拡張は保持するが、アルファの評価軸には含めない。  
まずは「操作して楽しい」「帰るまで楽しい」の2点を証明する。
