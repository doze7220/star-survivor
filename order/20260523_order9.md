【1. 実装調整：マイクロHUDの円弧配置】
現在、レーダー内のテキストがマーカーと重なって視認性が悪いため、円弧に沿って描画させます。
以下のヘルパー関数を `draw()` 内（またはグローバル）に追加し、これを使って「レベル＆経験値」および「スピード＆Gフォース」を描画してください。

// ▼ 追加するヘルパー関数
function drawCircularText(ctx, text, radius, startAngle, isBottom = false) {
    ctx.save();
    ctx.textAlign = 'center';
    ctx.textBaseline = 'middle';
    
    let totalAngle = 0;
    for (let i = 0; i < text.length; i++) {
        totalAngle += ctx.measureText(text[i]).width / radius;
    }
    
    let currentAngle = startAngle - (totalAngle / 2);
    ctx.rotate(currentAngle);
    
    for (let i = 0; i < text.length; i++) {
        const char = text[i];
        const charWidth = ctx.measureText(char).width;
        const charAngle = charWidth / radius;
        
        ctx.save();
        if (isBottom) {
            ctx.rotate(charAngle / 2);
            ctx.translate(0, radius);
            ctx.rotate(Math.PI); 
        } else {
            ctx.rotate(charAngle / 2);
            ctx.translate(0, -radius);
        }
        ctx.fillText(char, 0, 0);
        ctx.restore();
        
        ctx.rotate(charAngle);
    }
    ctx.restore();
}

＜適用方法＞
- 上部（レベル等）は `startAngle` を `-Math.PI / 2` (12時方向)、`isBottom = false` で描画。
- 下部（速度等）は `startAngle` を `Math.PI / 2` (6時方向)、`isBottom = true` で描画。
- Gフォースメーターの前には「ACC: 」を追加してください（例: `ACC: 3.2 G`）。
- 速度とGフォースは、半径（radius）を少しズラして2行になるように描画してください。
