# 『STAR SURVIVER』アルファ版実装指示書

## 1. プロジェクト概要
- コンセプト: 見下ろし型宇宙空間サバイバル・シューティング
- 操作感: レトロ（慣性強め）× 現代的（全方位射撃）
- 主要キャラ: シエロ・ルマスター（オペレーター）

## 2. フィールド・座標設計
- 拠点（アンカーガレージ）: 宇宙座標 (0, 0) 固定
  - アルファ版：被弾判定なしの無敵発着場。
- 敵スポーンポイント: 宇宙座標 (-500, -1500)
- リスポーン制御ルール:
  - プレイヤーの Y座標 <= -500 の時のみ新規リスポーン実行。
  - Y > -500 では増援停止。ただし「既に湧いている敵」は執拗に追跡してくる。

## 3. 自機挙動・戦闘
- 慣性移動: 宇宙空間特有の「滑る」感覚を重視。
- 360度砲塔: 移動方向に関わらず、マウス/右スティックで独立した照準が可能。
- 着艦条件: カタパルト上で自機の速度ベクトルが ±5 以下で成功。

- 補足
    - 砲塔は機体腹の部分に設置イメージ。細長い白の矩形で表示で、船首から少しはみ出るぐらいのサイズ。翼の下には隠れてしまうが許容。
    - 砲塔が向いている方向をレーダーに表示。内側から▲のアイコンで表示。三角の底辺に沿うように TARGET 表記。文字は三角に追従して回転。

# 🚀 【追加】ミサイル・ブースター詳細仕様書

## 1. 誘導ミサイル（CTRLキー）
- ロックオン挙動: 砲塔（マウス）が向いている方向、前方約90度の範囲内の敵を自動捕捉。
- 弾道: 自機の慣性を引き継いで射出後、放物線を描きながらターゲットへ誘導。
- バランス: 旋回性能に限界を設け、敵の急旋回で振り切られる可能性を残す。

## 2. 高出力ブースター（SHIFTキー）
- 基本挙動: 発動時、機首が向いている方向へ瞬間的に強大な加速度を付与。
- 慣性リセット: 発動時、それまでの横滑りベクトルを進行方向へ50〜70%修正（上書き）。
- 制限: 使用後は数秒の冷却時間（オーバーヒート）が必要。

## 3.補足
- パラメータ類に関しては、一時おまかせ

# 通信演出（シエロ・ナビゲーション）
- エースコンバット風通信表示、テキストのみ。色はグリーン。フォントサイズは18px。位置は画面上部中央・テキストセンタリング。
- メッセージの表示時間目安は、文字数x1秒。
- ”＝”で囲む。 例：＝　3, 2, 1, GO!　＝
- 出撃時: 「3, 2, 1, GO!」
- 索敵時: 「10時にターゲット発見ですー！」
- 特殊兵装使用時: 「ミサイル、いってらっしゃーい！」「ブースト全開ですね！」
- ミッション達成: 「ノルマ達成ですー、無理せず帰ってきてくださいね」
- 帰還(着陸完了時): 「おかえりなさい、お疲れ様でした！」
- プレイヤーHP20%以下 : 「無理しないで、帰還してください！」
- ゲームオーバー時: 「……傭兵さーん！」

- 補足
    - ミッション達成は敵10体撃破

# 母艦(拠点) アンカー・ガレージ
- 右MotherShip→Anker Garageに名称変更
- 上向きでマップに設置
- サイズは「カタパルトが自機と同じ幅」になるように。
- 着陸可能ポイントは、カタパルトと同矩形
- デザイン仕様書 spec_star_suvivor_mothership_design.md

# ミッションクリア
- ミッションクリアフラグ：敵を10機撃破
- 拠点に帰還し、着陸したらクリアとする。
- 着陸したらリザルト「MISSION CLEAR」を表示。それ以外はゲームオーバー時と同じダイアログでOK。リトライで最初から。


# その他調整
- HPゲージ、ヒートゲージ、EXPゲージは、左右で完全な円にせず、右10時～8時、左2時～4時。
- 右側ゲージ 内側から HPゲージ、ヒートゲージ
- 左側ゲージ 内側から EXPゲージ
- クレジット（懸賞金）表示
    - 画面左上に獲得懸賞金を表示
    - 敵１体ごとに100カウントUP、瞬時に足すのではなく、5刻みで目標値までカウントアップ。
