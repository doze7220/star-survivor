# 役割
君はVANGUARDRIFTER（HTML5/Canvasベースのスペースシューティングゲーム）のリードプログラマーだ。
既存のコード（v0.4.00想定）を基に、以下の要件に従ってHUDおよびレーダーUIの拡張（v0.5.00）を実装せよ。

# 実装要件：v0.5.00 コクピットUI統合改修

## 1. プロジェクト設定と全体調整
- バージョン表記を "v0.5.00" に更新せよ。
- UI要素が既存のグラフィックに埋もれないよう、新規テキストには適切な背景色や縁取り（ストローク）を適用し、Zオーダー（描画順）を整理すること。

## 2. ミニマップへの情報集約（座標表示の移行）
- 現在レーダー下部に表示されている自機座標テキスト（`- X.Y -`）を削除せよ。
- 削除した座標情報を、画面右上の「ミニマップ」のコンテナ内（下部中央または左下隅）にオーバーレイ描画せよ。
- 視認性確保のため、フォントサイズは10px程度とし、テキストの背景に半透明の黒帯を敷くか、テキスト自体に強い縁取りを行うこと。

## 3. レーダー内マイクロHUDの拡張
画面中央の自機追従レーダー（RADAR_RADIUS）の内側および周辺に、以下の情報を追加描画せよ。

- **A. レベル＆経験値表示（上部）**
  - レーダー真円の内側上部（10時〜2時の円弧沿い、または上部中央）に、現在のレベルと経験値を `Lv.XXX Exp.xxxx/xxxx` のフォーマットで描画せよ。
  - 右上HUDを削除した代替として、プレイヤーが常に成長状況を視認できるようサイバーパンク風のフォントで配置すること。

- **B. スピード＆Gフォースメーター（下部）**
  - レーダー真円の内側下部（4時〜8時の円弧沿い、または下部中央）に、以下の2つの数値を描画せよ。
  - ① **速度**: 現在の自機速度（`km/s`）。
  - ② **加速度 (G)**: 1フレーム前からの速度ベクトル変化量（`Math.hypot(player.vx - prevVx, player.vy - prevVy)`）を計算し、任意の係数を掛けて「Gフォース」として算出・表示（例: `3.2 G`）すること。急旋回やブレーキ時のG負荷を視覚化する狙いである。

- **C. ダイナミック・コンテクスト・アラート（状況警告）**
  - 自機の状態に応じ、レーダー中央（自機アイコンの直上など）に警告テキストを動的に明滅表示させるロジックを追加せよ。
  - [被弾時 or 敵機超接近時]: `WARNING` (赤色)
  - [オーバーヒート時]: `OVERHEAT` (オレンジまたは灰色)
  - [帰艦可能状態]: `DOCK INBOUND` (緑色)

## 4. ターゲット情報の拡張
- レーダー外周に表示される「画面外の敵方向マーカー（赤い▲）」付近の情報を拡張せよ。
- 現在の「距離（m / km）」表示に加えて、対象の敵の「残りHP（パーセンテージ、または極小のHPバー）」を併記し、画面外の敵の損傷度合いを把握できるようにすること。
