# 作業指示書：物理法則と同等性の統合 (Newtonian Symmetry Refactoring)

## 目的
自機（プレイヤー）と敵機（AI）の物理挙動、入力処理、および衝突判定を完全に統一し、物理STGとしてのフェアネスを担保する。敵機に対する「座標や速度の直接的な操作（ズル）」を一切禁止し、プレイヤーと全く同じ制約の中で動作させること。

## 実装ステップと要件

### 1. 入力インターフェースの仮想化 (Virtual Control & RCS Ready)
すべての機体（`player`, `enemies`）は、機体制御用の仮想入力オブジェクト `inputs` を介してのみ機動を行うように改修する。
将来的な「多方向スラスター（RCS/補助バーニア）」搭載機体を見据え、単純なキーのON/OFFではなく、並進力と回転力を意識した入力構造を準備すること。

- **データ構造の定義:** `inputs: { thrustX: number, thrustY: number, yaw: number }` （値は -1.0 ～ 1.0）
  - **現在の標準機体の場合:** Wキーは `thrustY = 1.0`（メインスラスター前進）、A/Dキーは `yaw = -1.0 / 1.0`（姿勢制御）としてマッピングする。
  - **将来の補助スラスター機体想定:** テンキー1〜9等で全方位への平行移動（ストレイフ）を行う場合、`thrustX`（左右推力）やマイナスの `thrustY`（逆噴射）を利用する。
- **プレイヤー:** キーボード入力を上記の `inputs` パラメータに変換して渡す。
- **敵機 (AI):** `getAIInput(enemy, target)` 関数を実装し、ターゲットとの相対座標から必要な推力（`thrustX, thrustY`）と旋回（`yaw`）を算出して `inputs` を生成する。

### 2. 物理演算エンジンの単一化 (Unified Physics)
現在の `player` 用と `enemies` 用の移動ロジックを完全に廃止し、共通関数 `applyPhysics(entity, inputs, stats)` を実装する。

- **旋回:** `inputs.yaw` と `stats.handling` に基づき機体の `bodyAngle` を更新する。AIロジック内での直接的な角度代入は固く禁ずる。
- **加速（メインおよび補助スラスターの統合処理）:** - `inputs.thrustY` と `stats.moveSpeed` に基づき、現在の機体の正面方向（`entity.bodyAngle`）への推力ベクトルを加算する。
  - `inputs.thrustX` と `stats.moveSpeed`（または将来追加される補助スラスター用係数）に基づき、機体の側面方向（`entity.bodyAngle` の直交方向）への推力ベクトルを加算する。
  - これにより、「正面を向いたまま真横や斜めに加速する」挙動が物理エンジンの追加改修なしで実現可能となる。
- **環境物理:** 宇宙空間の摩擦（`CONFIG.FRICTION`）と最高速度のクランプ処理は、全機体共通で適用すること。

### 3. 衝突・ダメージ処理の対称化 (Symmetric Collision)
現在の「自機が敵にぶつかった場合」「敵同士がぶつかった場合」などの個別処理を廃止し、共通関数 `resolveCollision(entityA, entityB)` を実装する。

- **反動（Recoil）:** 衝突法線ベクトルと相対速度（v_rel）から、反発係数（`CONFIG.COLLISION_RECOIL_RESTITUTION`）を用いた跳ね返りベクトル（力積）を計算し、`entityA` と `entityB` の双方に等しく適用すること。
- **ダメージ:** 衝突時の相対速度の大きさ（クラッシュスピード）に比例するダメージを計算し、双方のHPから全く同じダメージ量を減算すること。

## 完了条件
- コード内から敵機専用の非物理的な移動ロジック（直接的な座標計算や角度の書き換え）が完全に消去されていること。
- 敵・味方問わず、全ての推力と旋回が `inputs` の `thrustX/Y`, `yaw` を介して物理エンジン (`applyPhysics`) に渡され、処理されていること。
- 多方向スラスターの処理基盤（直交推力の加算ロジック）が `applyPhysics` 内に準備されていること（現時点でそれを利用する機体がなくても実装に含める）。
- いかなる衝突においても、当事者双方が運動量とダメージを等しく引き受けていること。
