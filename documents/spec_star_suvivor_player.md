1. 基本ステータス定義

2. 防御属性および装甲耐性
ダメージ計算は「基礎ダメージ量 × (1 - 各属性耐性率)」で行う。機体特性により以下の3種の耐性が重要となる。
衝撃耐性（Impact）
物理弾、隕石、敵機体との衝突ダメージを軽減。レトロシューティング特有の事故死を防ぐ重要な指標。
熱耐性（Heat）
レーザー兵器、恒星フレア、プラズマ攻撃を軽減。シールド性能や耐熱塗装に依存する。
爆破耐性（Blast）
ミサイル、魚雷、およびその爆風ダメージを軽減。範囲攻撃に巻き込まれた際の生存率に直結する。
3. 初期機体バリエーション仕様
各機体は、初期ステータスだけでなく、物理挙動や武器運用に固有の特性を持つ。
3.1. 強襲型：バルカン特化機
コンセプト: 圧倒的な弾幕による制圧力を持つ、実弾兵器運用のエキスパート。
推奨プレイスタイル: 高い加速度を活かしたドッグファイト。
固有特性:
反動制御: バルカン射撃時の自機への反動（Recoil）を50%軽減。
衝撃ボーナス: 実弾系武器の衝撃耐性貫通 +15%。
ステータス補正: 加速度 +20% / 衝撃耐性 +15%。
初期装備: ヘビーバルカン（1発あたりの蓄積熱量：2.0）。
3.2. 精密型：レーザー特化機
コンセプト: 冷却性能とエネルギー収束率を極限まで高めた光学兵器特化機。
推奨プレイスタイル: 一点集中照射による高耐久標的の迅速な排除。
固有特性:
冷却加速: 射撃停止から冷却開始までのディレイを0にする。
貫通仕様: 初期装備レーザーが敵を1体貫通する。
ステータス補正: 冷却速度 +40% / 熱耐性 +20%。
初期装備: フェーズレーザー（持続照射中、毎秒15.0の熱を蓄積）。
3.3. 重装型：ミサイル特化機
コンセプト: 移動要塞としての性能を追求し、広範囲を壊滅させる重武装機。
推奨プレイスタイル: 低機動力を高い装甲で補い、複数のミサイルで敵集団を殲滅。
固有特性:
爆破拡張: 弾着時の爆発半径を1.5倍に拡大。
ノックバック耐性: 被弾時の慣性移動の乱れを80%カット。
ステータス補正: 最大HP +60% / 爆破耐性 +30% / 最大速度 -25%。
初期装備: Homingミサイル（1発あたりの蓄積熱量：12.0）。
3.4. 指揮型：ビット特化機（空母）
コンセプト: 複数の自律攻撃ユニット「ビット」を制御する戦術指揮機。
推奨プレイスタイル: 自身の回避に専念し、ビットによる自動迎撃で敵を排除。
固有特性:
自律制御: ビットは射程内の最も近い敵を自動追尾して射撃する。
同時運用: 初期ビット稼働数 +2基。
ステータス補正: ヒートゲージ容量 +20% / 本体攻撃力 -30%。
初期装備: アサルトビット（ビットの蓄積熱は本体のゲージを共有）。
3.5. 偵察型：高機動スカウト
コンセプト: 極めて高い機動性と情報収集能力を備えた高効率機。
推奨プレイスタイル: 広大なマップを高速移動し、リソースを独占して高速成長する。
固有特性:
高性能レーダー: 画面外のボス、宝箱、レアアイテムの方向をUIエッジにアイコン表示。
探査ビット: アイテム回収に特化したビット。本体から離れた位置のアイテムも自動回収。
ステータス補正: 最大速度 +40% / 加速度 +30% / トラクタービーム範囲 +150%。
初期装備: スカウトレーザー（低火力だが射程が長い）。
4. 成長・パワーアップの拡張性
レベルアップに伴う成長は、宇宙空間での物理法則と現代的なビルド構築を融合させた形式で提供する。
レベルアップ選択（アップグレード） 経験値収集によるレベルアップ時、ランダムに提示される3〜4枚のカードから1つを選択。「最大速度上昇」や「ヒート容量拡張」などの基礎能力向上、または新規武器の取得を行う。
武器進化（Evolution） 武器レベルを最大化し、かつ特定の補機（パッシブアイテム）を所持している場合、武器は「進化」する。進化後は攻撃パターンが劇的に変化するだけでなく、熱蓄積効率の大幅な改善、あるいは「衝撃属性と爆破属性の複合化」などの強力な特性が付与される。
慣性挙動のカスタマイズ 成長要素には「摩擦係数の低下（より滑るが高速）」や「スラスト力の極大化」など、プレイヤーの操作スキルに合わせた物理パラメータ調整項目も含まれる。これにより、レトロシューティング特有の「乗りこなす楽しさ」を拡張する。
